# Creando Planes de Internet

Los **Planes de Internet** definen la velocidad, precio y características del servicio que ofreces a tus clientes.

## Tipos de Planes

En uManager puedes crear diferentes tipos de planes:

- **Residencial**: Para hogares, generalmente con reuso.
- **Corporativo**: Para empresas, con ancho de banda garantizado.
- **Fibra Óptica**: Planes específicos para tecnología FTTH.
- **Inalámbrico**: Planes ajustados para enlaces por aire.

## Pasos para crear un plan

1. Ve a **Settings > Planes de Internet**.
2. Haz clic en **"Nuevo Plan"**.
3. Completa los detalles:
    - **Nombre del Plan**: Ej: "Fibra 50MB".
    - **Velocidad de Bajada (Download)**: En Mbps o Kbps.
    - **Velocidad de Subida (Upload)**: En Mbps o Kbps.
    - **Precio Mensual**: El valor a cobrar al cliente.
    - **Reuso (Overbooking)**: Define la garantía del ancho de banda (1:1, 1:4, etc.).

## Configuración Avanzada

Puedes configurar opciones adicionales como:

- **Burst Limit**: Velocidad máxima temporal (Ráfaga).
- **Burst Threshold**: Umbral para activar la ráfaga.
- **Burst Time**: Duración de la ráfaga.

> [!TIP]
> Configurar correctamente el Burst permite que la navegación web se sienta más rápida sin saturar tu red constantemente.

## Asignación a Clientes

Una vez creado, el plan estará disponible para ser asignado a cualquier cliente nuevo o existente desde la ficha del cliente.
