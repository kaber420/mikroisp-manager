# Creando tu Primera Zona

Las **zonas** son la forma de organizar tu red geográficamente. Cada zona puede contener múltiples Access Points y clientes.

### ¿Por qué usar zonas?

Las zonas te permiten:

- Organizar tu red por áreas geográficas (barrios, sectores, ciudades)
- Asignar técnicos específicos a cada zona
- Facilitar el mantenimiento y troubleshooting
- Generar reportes por zona

### Pasos para crear una zona

1. **Acceder al módulo de Zonas**: Haz clic en **Zones** en el menú lateral.
2. **Crear nueva zona**: Haz clic en el botón **"+ Nueva Zona"**.
3. **Completar la información**:
    - **Nombre**: Un nombre descriptivo para la zona (ej: "Centro", "Barrio Norte")
    - **Descripción** (opcional): Detalles adicionales sobre la zona
    - **Color** (opcional): Un color para identificar la zona visualmente
4. **Guardar**: Haz clic en **"Guardar"**.
